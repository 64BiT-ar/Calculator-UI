# \# JS Calculator

# 

# \## Overview

# JS Calculator is a simple and responsive web-based calculator built with HTML, CSS, and JavaScript. It provides a clean interface for performing basic arithmetic operations while displaying both the current equation and the calculated answer.

# 

# \## Features

# \- Basic arithmetic operations: addition, subtraction, multiplication, and division

# \- Live equation display while entering expressions

# \- Backspace support for quick corrections

# \- Clear-all function to reset the calculator

# \- Responsive layout with a modern dark interface

# 

# \## Technologies Used

# \- HTML5

# \- CSS3

# \- JavaScript

# 

# \## Project Structure

# \- index.html - Main calculator interface

# \- style.css - Styling and layout

# \- script.js - Calculator logic and interactions

# 

# \## Getting Started

# You do not need any build tools or package installation for this project. It runs directly in a browser.

# 

# \### Steps

# 1\. Clone or download the project files.

# 2\. Open index.html in your browser.

# 3\. Use the on-screen buttons to enter calculations.

# 

# \## How It Works

# \- Press number and operator buttons to build an expression.

# \- The equation is shown in the top display.

# \- Press = to evaluate the expression and show the result.

# \- Use AC to clear everything or the backspace button to remove the last character.

# 

# \## Notes

# \- The calculator is designed for basic arithmetic use cases.

# \- Expressions are evaluated directly from the entered input, so only valid arithmetic input should be used.
